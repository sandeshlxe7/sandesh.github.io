A computer is an electronic machine that receives data (input), processes it using hardware and software, and gives output. It can store data, perform calculations, and run programs.

  • RAM: Temporary memory used while the computer is running.

  • Power Supply: Gives power to all components.
  • GPU: Graphics card for videos, games, and rendering.
